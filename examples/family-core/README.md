# family-core Example

This directory is reserved for a minimal example that shows how a client deployment can consume the `family-core-minimal` reference profile.

Current example material:
- `rauthy.toml.template` — example Rauthy config template with placeholder secrets
- `rauthy.env.example` — example runtime env file shape for the auth baseline
- `step-ca.json.template` — example step-ca config shape for the private CA baseline

# Templates

This directory contains reusable templates and starter defaults for downstream consumption.

The initial focus is:
- a copy-oriented starter default for `family-core` foundation services;
- a minimal client deployment template for `family-core`.

Together they provide:
- a vendor-shipped seed for baseline service configuration;
- the shared site-domain contract;
- a runtime secret materialization example for `rauthy.env`;
- opt-in placeholders for the private DNS / CA / ingress path.

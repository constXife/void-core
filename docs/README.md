# Documentation

This directory contains the canonical documentation for the public foundation baseline of `void-core`.

If you are new here, read the documents in this order.

## Start here

- `TRUST_MODEL.md` — what `void-core` is meant to protect, and what it must not own
- `PUBLIC_SCOPE.md` — the explicit allowlist-style scope of what may live in the public foundation
- `FOUNDATION_COMPONENTS.md` — the current foundation building blocks, module scaffold, and their boundaries

## Component and extension docs

- `ATRIUM.md` — the role and limits of the Atrium foundation shell
- `../platform/web-host-contract/README.md` — the canonical contract for session-aware private web hosts
- `CA_ROTATION.md` — the canonical PKI rotation model for the `step-ca` baseline

## Language

Canonical documentation in `void-core` is English-first.

Translations may be added later, but the English version remains the source of truth unless explicitly stated otherwise.
